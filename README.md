# Number Guessing Game

CMPSC 132 Final Project by Abdullah Moquim

## Project Description

This project is a Number Guessing Game created in Python. The player chooses a difficulty level and tries to guess the secret number within a limited number of attempts. After each guess, the program gives hints by telling the player if the guess is too high or too low.

## Features

- Easy, Medium, and Hard difficulty levels
- Different number ranges based on difficulty
- Limited attempts depending on the selected level
- Too High / Too Low hints
- Attempts counter
- Performance message after each win
- Replay option
- Session statistics:
  - Games Played
  - Wins
  - Best Score

## How to Run

1. Download the project files
2. Open the file in Python
3. Run `number_guessing_game.py`
